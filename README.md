# 🧮 KidsMathCalculations (5岁幼童专属·奇妙数学冒险 ✨)

专为 **5岁左右学龄前至小学一年级儿童** 量身定制的极简视觉化、情境化数学启蒙 Web 应用程序。采用精美梦幻的“糖果粉拟物玻璃（Candy Glassmorphism）”设计系统，搭配活泼的弹簧缓动微动画、全套童音朗读和无挫败感纠错机制，打造沉浸式学习乐园。

---

## ✨ 核心亮点

### 1. 🌈 15 大交互式数学启蒙游戏（三大分类）
系统性地将低幼思维启蒙划分为三大认知板块，共包含 15 个精心打磨的游戏化关卡：
* **🔢 数感与数字**：
  * **比大小 (Compare)**：从个位数比较，逐步进阶到算式与算式的逻辑天平。
  * **数字排序 (Sort)**：动态拖拽数字或算式，培养数轴顺序感知。
  * **数列填空 (Fill Sequence)**：跳数规律（正数、倒数、倍数）数列填空。
  * **凑十法 (Make Ten)**：运用特制 **蒙特梭利黄金珠棒（Golden Bead Bars）** 形象练习补数。
  * **破十减法 (Teens Subtraction)**：**[NEW]** 专为退位减法设计的“圈小棒法”、“树状拆分法”与“想加算减气泡”，配以 **全套破十法口诀魔法卡**，让抽象退位变直观。
  * **乘法启蒙 (Intro Multi)**：通过排列组合图解乘法本质，支持挑战趣味文字题。
* **🔷 图形与空间**：
  * **认形状 (Shapes)**：认识基本几何平面图形，高阶关卡融入文字谜题。
  * **找规律 (Patterns)**：ABAB、AABB、ABCABC 动态交替，强化空间序列数感。
  * **数形状 (Count Shapes)**：在复杂混杂图形中分类点数，高阶融入数量差逻辑。
  * **空间方位 (Spatial Sense)**：上下左右、内外方向的情境拼图。
* **🌈 综合生活认知**：
  * **时钟练习 (Time Clock)**：互动旋转时针与分针，整点、半点、一刻钟情理连结。
  * **认颜色 (Colors)**：三原色认知与趣味混色魔法实验。
  * **认星期 (Weekdays)**：感知星期循环与“昨天、今天、明天”的时间跨度。
  * **认季节 (Seasons)**：探索四季景色更替、月份归属与自然常识。
  * **购物启蒙 (Shopping)**：模拟收银付款、选商品与零钱找零，生活数学大实操。

### 2. 🧠 混合动力双轨难度系统 (Hybrid Difficulty)
打破传统死板分级，引入**“家长全局锁定 + 智能自适应”**混合机制：
* 🌟 **智能自适应 (Smart Adaptive)**（推荐）：答对自动进阶（Level 1 ➔ 2 ➔ 3），答错自动温柔降级或复习，匹配孩子即时认知边界。
* 🟢 **基础 / 🟡 进阶 / 🔴 挑战**：家长可在“教案配置室”一键强制锁定，因材施教。

### 3. 💡 科学育儿·教育理念弹窗 (Why We Play & Read)
* **[NEW]** 顶栏新增 `💡 理念` 按钮，内置精美高清的 **《思维启蒙核心痛点：逻辑理解，数学开窍的隐形地基》** 纵向长图海报。
* 向家长传递：*不急着“刷题”，而是帮孩子“开窍”！阅读理解才是数学的“隐形地基”*。通过玩桌游、看绘本、听指令、讲规则，给孩子培养数据预处理与因果空间推演能力。

### 4. ⚙️ 家长专属控制台 (Parental Dashboard)
* **家长锁 (Guardian Gate)**：需要正确解答两位数算术题才允许进入，杜绝幼儿误触。
* **自定义计算范围**：可自由设定 10以内、20以内、50以内 或 1-9999 的任意个性化数值界限。
* **学习阶段管理器 (Stage Control)**：分四个层级控制拖拽教具（草莓、珠棒）的显示：
  * *阶段一：感知计数（强显教具）* ➔ *阶段二：具象加减* ➔ *阶段三：半抽象（按需点击显示）* ➔ *阶段四：心算纯计算（无教具）*。
* **跨设备进度同步 (Progress Sync)**：提供一键 Base64 压缩编码“同步码”，轻松在 iPad、电脑和手机间互传错题本与教案设置。

### 5. 🧸 温柔无挫败设计 (Frustration-Free)
* 答错时**无刺耳警报**，不留心理阴影。连续错题时，实物图形会温和发光呼吸，语音柔和提示“数一数上面的小草莓吧~”，手把手引渡难关。
* **智能童音朗读 (TTS)**：自动双语朗读题目，即使是不认字的 5 岁孩子也能无障碍独立练习。

## 🛠 开发与部署指南

本项目基于 **React + Vite + Vanilla CSS** 构建，精简轻量，零后端，易扩展。

### 1. 本地运行

```bash
# 1. 安装依赖
npm install

# 2. 启动 Vite 本地开发服务器
npm run dev
```

* 本地服务默认启动于：`http://127.0.0.1:3000/`

### 2. 构建与部署到 GitHub Pages

我们配置了 `gh-pages` 脚本，可实现一键编译并发布至生产分支：

```bash
# 一键编译并部署到 GitHub Pages 分支 (gh-pages)
npm run deploy
```

> **部署注意事项：**
> * 编译基础路径已在 `vite.config.js` 中完美配置：开发环境为 `/`，生产环境自动匹配仓库名子路径 `/KidsMathCalculations/`。
> * 线上访问地址：`https://grenvill-create.github.io/KidsMathCalculations/`
