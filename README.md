# 🧮 KidsMathCalculations (5岁幼童专属版)

专为 **5岁左右的学龄前儿童** 设计的极简视觉化算术启蒙 Web 应用程序。

## ✨ 核心特色

1. **跨平台 PWA 免安装运行**：
   - 完美适配 PC 浏览器。
   - 在手机（Android/iOS）浏览器中打开网页，点击“添加到主屏幕”，即可像原生 App 一样全屏、离线运行，无任何广告干扰。
2. **家长专属控制台 (Parental Dashboard)**：
   - 特设“家长锁（Guardian Gate）”，需完成成人难度的算术题方可进入设置。
   - **完全自定义数值范围**：摒弃死板的难度分级，家长可自由设定数字范围（如：10以内、20以内、50以内），系统生成的题目绝不会超出此上限。
   - **一键屏蔽减法**：针对启蒙初期的儿童，可隐藏减法专精加法练习。
3. **实物点数引导**：
   - 屏幕上显示具体的“草莓”、“星星”等实物图形辅助计算。
   - 幼儿可以直接点击屏幕上的图形，会伴有“叮当”声并放大，帮助建立“点数”的概念。
4. **温柔的错误引导 (Frustration-Free)**：
   - 答错时不会出现刺耳警告音或吓人的红色大叉。
   - 连续答错两次时，系统会用温和的童音提示“数一数上面的图形吧！”，并且对应的实物图形会发光脉冲，手把手引导孩子。
5. **智能语音朗读 (Text-to-Speech)**：
   - 出题时自动用童音朗读“三加二等于几？”，让不识字的5岁儿童也能独立操作。

## 🛠 开发与部署指南

本项目基于 **React 19 + Vite** 构建，零后端配置。

### 1. 本地运行

```bash
# 安装依赖
npm install

# 启动本地开发服务器
npm run dev
```

### 2. 构建与部署到 GitHub Pages

我们使用 `gh-pages` 进行一键自动化部署。

**每次修改代码后，请按以下步骤更新到线上：**

```bash
# 1. 提交源代码修改
git add .
git commit -m "update code"
git push origin main

# 2. 一键编译并部署到 GitHub Pages
npm run deploy
```

> **注意：** 部署的地址取决于 `vite.config.js` 中的 `base` 路径配置。当前配置的地址应为：`https://grenvill-create.github.io/KidsMathCalculations/`
