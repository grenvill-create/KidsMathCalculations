# KidsMathCalculations Workspace Rules

## Deployment
- Whenever you finish making code modifications, implementing new features, or fixing bugs for this project, you MUST automatically commit the changes to Git and push/deploy them to GitHub.
- Use `git add .`, `git commit -m "..."`, `git push`, and `npm run deploy` to accomplish this.
- Do not wait for the user to explicitly ask you to update GitHub. Make it an automatic final step of your implementation workflow.
